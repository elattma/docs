---
title: "Build a Library"
---

1. Navigate to the [Library](https://mimo.team/dashboard/library) tab 
2. Press create a connection to add a data source
3. Select the data source you want to integrate
4. Choose your preferred authentication method (OAuth, API credentials, etc.)
5. Authenticate and name your connection
6. Wait a few moments while Mimo ingests and makes your data AI ready before [querying](/api-reference/endpoints/context/get)
